## Install

```bash
# Install a skill directly from this repo
hermes skills install https://github.com/Codek365/hermes-skills/raw/main/social-media/hermes-tweet/SKILL.md
hermes skills install https://github.com/Codek365/hermes-skills/raw/main/browser-automation/cmux-browser-4-hermes-ai/SKILL.md

# Or add this repo as a skill source (tap)
hermes skills tap add Codek365/hermes-skills
```
